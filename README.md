# Komukai Quest（小向クエスト）

T芝K向工場を舞台にした、風刺的ベルトスクロール格闘アクション。
「小向戦士」となり、数々の困難と裏切りを乗り越えて戦士の頂点を目指せ。

**▶ トップページ: https://naokianraku.github.io/komukai-quest/** （「ゲームをする」「遊び方を読む」へ分岐）
**▶ ゲーム直リンク: https://naokianraku.github.io/komukai-quest/game/**

> 🎮 遊ぶ人向けの説明は **[あそびかた.md](あそびかた.md)**（サイト上は [あそびかた.html](あそびかた.html)）へ。この README は開発者向け。

- **依存ゼロ**: HTML5 Canvas + 素のJavaScript（ES Modules）。ビルド不要・静的サイト。
- **ピクセルアート**: ドット絵をコード生成。高DPI描画でくっきり。
- **2.5D ベルトスクロール**: ファイナルファイト型。奥行き(y)＋ジャンプ(z)。
- **BGM＋効果音**: WebAudioで生成するチップチューン（外部ファイルなし）。
- PC・スマホ対応（タッチUI自動表示）。

## 操作

| 操作 | キー | スマホ |
|---|---|---|
| 移動（4方向 / 奥行き） | ← ↑ ↓ → / W A S D | 左半分のタッチした位置が中心になるフローティング・スティック（待機時は左下に薄いリング） |
| 攻撃（連打でコンボ） | J / Z | 右端下の「攻撃」ボタン |
| ジャンプ | K / X | 「攻撃」の真上「跳」ボタン |
| 一時停止 | P / Esc | 画面上部 ❚❚ ボタン |
| BGMミュート | M | タイトルの「BGM♪」ボタン |
| 決定（メニュー） | Enter / Space | タップ |

※ ブラウザの自動再生規制のため、BGMは**最初の操作（キー/タップ）で鳴り始めます**。

## 世界観

- **敵**: 上司・部下・客・調達先ベンダー・スタッフ部門、各種「審査会」「管理チーム」など社内外の関門。
- **味方（PJメンバ）**: 各ステージに3人。だが**ボスの直前まで来ると全員が裏切って敵に回る**。さらに戦闘中の**うっかりミスで後ろから誤射**してくることも（信用できるのは自分だけ）。

### 言葉の暴力
敵は攻撃や雑談で、頭上の吹き出しに「言葉の暴力」を浴びせてくる（例: 「全体像が見えない！」「再DRだな」「該非判定はどこだ？」「机叩かないで価格交渉できるのか」）。役職別の台詞は `セリフ一覧.md`（ローカル）→ `src/taunts.js` に生成。

### 識別
- **作業着は全員水色＋黒の安全靴**（現場/技術/スタッフ/管理職）。**客・ベンダー・関連会社社長はスーツ**。
- ヘルメット色＋**頭上の役職ラベル**で識別。自分は**橙の▼マーカー**、現在地はHUDに表示。

## 難易度
- **ふつう**: 標準ルール（残機3）。
- **かんたん**: 残機多め（残機6）＋**自分も飛び道具**（攻撃で工具を投げる）。
- **地獄**: **雑魚敵が2倍**に増殖（中ボス・部長級・ボスは増えない）＋残機少なめ（残機2）。

※ プレイヤーの体力ゲージは「**仕事体力**」。0になると残機が減る。

## ステージ（出世）
小向戦士 → 小向小隊長（主務級）→ 小向大隊長（EX級）→ 小向司令官（課長級）

| Stage | 現在地 | プレイヤー | ボス | 主なギミック |
|---|---|---|---|---|
| 1 | 60号7階Cゾーン（技術部） | 小向戦士 | 技術部長 | 設計審査（ランクB）5人の徒党 |
| 2 | 60号7階Dゾーン（技術管理部） | 小向小隊長 | 技師長（＋技術管理部長が同時出現） | 輸出管理チームの徒党／**中ボス 管理担当K** |
| 3 | 60号5階（生産管理・生産企画） | 小向大隊長 | 工場長（＋生産部長が同時出現） | **中ボス 生産企画課長**／PMR審査会（部長5名）／**「出向」の扉→関連会社(62号6階)** |
| 4 | 本社 事業部フロア | 小向司令官 | 事業部長（＋企画部長が同時出現） | 最終決戦 |
| 裏 | 60号7階Aゾーン（業火） | 小向司令官 | 魔王Y下 | **🔥 ネタバレ注意**：事業部長を倒した“その先”。Y下四天王＋魔王Y下（常軌を逸したパワハラ） |

- **PJメンバは、ボスが最初の一言（固定の決めゼリフ）を放った直後に全員裏切る**（一部は道中で先に寝返る）。
- 各ステージのボスは**一言目だけ固定フレーズ**（技術部長＝事前配布資料／技師長＝本番で全部回答しろ／工場長＝安全とコンプラ優先／事業部長＝売上・利益・納期・コスト）。2言目以降はランダム。
- 技術部長・生産部長・技術管理部長は**同じ部長系のセリフ**、事業部長・企画部長も**同じセリフ**を使う。
- **中ボス**（管理担当K／生産企画課長）と**裏ステージ**（魔王Y下／Y下四天王）のセリフは `セリフ一覧.md` の同名カテゴリに後から追記できる。
- **裏ステージ**：事業部長を撃破すると、業火に包まれた60号7階Aゾーンへ。**Y下四天王（N村／K目／S田／H本）**と魔王Y下を倒すとエンディング。N村＝生産企画課長・K目＝管理担当Kの再登場。S田とH本は攻撃してこない代わりに体力多めの“壁役”。
- **携行ミサイル兵**（誘導弾）はジャンプで回避可。
- Stage3の「出向」の扉に触れると関連会社（62号6階）へ出向し、社員・出向先上長・関連会社社長を倒すと本社へ帰還。

## 起動（ローカル）
ES Modules のため簡易HTTPサーバ経由で開く（`file://` 直開き不可）。リポジトリ直下で起動:

```bash
cd komukai
python3 -m http.server 8000     # もしくは: npx serve
# トップページ: http://localhost:8000/  ／ ゲーム直: http://localhost:8000/game/
```

## サイト構成
ルートはトップページ。ゲーム本体は `game/` 配下に独立（自己完結）。

```
index.html          トップページ（「ゲームをする」「遊び方を読む」へ分岐）
あそびかた.html      プレイヤー向けガイド（サイト表示用）
あそびかた.md        プレイヤー向けガイド（リポジトリ閲覧用・同内容）
.nojekyll           GitHub Pages を素の静的配信に（Jekyll無効）
game/
  index.html / style.css        ゲーム本体（/game/ で配信）
  src/
    main.js     ループ・状態機械・Stage・ベルトスクロール進行・出向/裏ステージ分岐
    config.js   定数・ユーティリティ
    input.js    キーボード＋タッチ（フローティングスティック/透過ボタン）
    audio.js    BGM・効果音（WebAudioチップチューン）
    sprites.js  ピクセルアート（作業着/スーツ・パレット・役職ラベル）
    entities.js Player / Char(敵・味方・ボス) / Projectile / 吹き出し
    combat.js   2.5D 当たり判定・エフェクト
    stages.js   敵/味方ステータス・ステージ定義（現在地・シーン・台詞カテゴリ）
    taunts.js   台詞コーパス（セリフ一覧.md から生成）
    ui.js       HUD・各画面・タイトル
```

> 台詞の正本 `セリフ一覧.md` と生成スクリプト `scripts/`（出力先 `game/src/taunts.js`）はローカル管理（リポジトリ非公開）。
