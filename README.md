# Komukai Quest（小向クエスト）

東芝・小向工場を舞台にした、風刺的ベルトスクロール格闘アクション。
「小向戦士」となり、数々の困難と裏切りを乗り越えて出世の頂点を目指せ。

- **依存ゼロ**: HTML5 Canvas + 素のJavaScript（ES Modules）。ビルド不要。
- **ピクセルアート**: ドット絵をコード生成。
- **2.5D ベルトスクロール**: ファイナルファイト型。奥行き(y)＋ジャンプ(z)。

## 遊び方（起動）

ES Modules を使うため、ローカルでは簡易HTTPサーバ経由で開く（`file://` 直開きは不可）。

```bash
cd komukai
python3 -m http.server 8000     # もしくは: npx serve
```

ブラウザで <http://localhost:8000/> を開く。

`naokianraku.github.io` などへそのまま置けば GitHub Pages で公開可（相対パスのみ・ビルド成果物なし）。

## 操作

| 操作 | キー |
|---|---|
| 移動（4方向 / 奥行き） | ← ↑ ↓ → または W A S D |
| 攻撃（連打でコンボ） | J または Z |
| ジャンプ | K または X |
| 決定（メニュー） | Enter / Space |

### スマホ（タッチ）
タッチ端末では自動で画面に操作UIが出る。**左下の仮想スティックで移動、右下の「攻撃」「跳」ボタンで攻撃・ジャンプ、メニューはどこでもタップで決定**。
（PCで確認したいときは URL に `?touch=1` を付ける）

## 世界観

- **敵**: 上司・部下・客・調達先ベンダー・スタッフ部門（人事/経理/品質保証）。
- **味方**: 同じプロジェクトの仲間だけ。**ただし裏切る者がいる**（要注意）。

### 言葉の暴力（敵の攻撃）
敵は攻撃モーションに合わせて頭上の吹き出しで「言葉の暴力」を浴びせてくる
（例: 「計画の全体像が見えない！」「報告が遅い！」「日割り工程を提出しろ！」
「前回よりコストが高い！説明しろ！」）。役職別の台詞は [src/taunts.js](src/taunts.js) に定義。

### 服装・識別
- **作業着は全員水色＋黒の安全靴**（現場/技術/スタッフ/管理職）。
- **客・調達先ベンダーはスーツ**（茶/紺＋ネクタイ）。
- ヘルメット色と**頭上の役職ラベル**（部下/先輩/スタッフ/客/ベンダー/味方/裏切り者）で識別。
- 自分（小向戦士）は**橙の▼マーカー**で常に分かる。

### 舞台
オフィスフロア／会議室がメイン。製造フロア・役員会議室も登場（ステージごとに切替）。

### 出世（プレイヤーのランク）
小向戦士 → 小向小隊長（課長級）→ 小向大隊長（部長級）→ 小向司令官（工場長超級）

### ボス
工場内の三大ボス **生産部長 → 技師長 → 工場長**、
そしてその上に立つ真のボス **事業部長**。

| ステージ | プレイヤー | ボス | 裏切り |
|---|---|---|---|
| 1 戦士編・現場 | 小向戦士 | 生産部長 | なし |
| 2 小隊長編・スタッフ部門 | 小向小隊長 | 技師長 | 他部署に引き抜かれ裏切る |
| 3 大隊長編・調達/渉外 | 小向大隊長 | 工場長 | 協力会社の手のひら返し |
| 4 司令官編・経営 | 小向司令官 | 事業部長（真のボス） | 最後の味方も裏切る |

## ソース構成

```
index.html / style.css
src/
  main.js       ループ・状態機械・Stage・ベルトスクロール進行
  config.js     定数・ユーティリティ
  input.js      キーボード入力
  sprites.js    ピクセルアート（水色作業着/スーツ・パレット・役職ラベル・描画）
  entities.js   Entity / Player(小向戦士) / Char(敵・味方・ボス) / Projectile / 吹き出し
  combat.js     2.5D 当たり判定・エフェクト
  stages.js     敵/味方ステータス・4ステージ定義（シーン・台詞カテゴリ）
  taunts.js     「言葉の暴力」台詞コーパス（役職別）
  ui.js         HUD・各画面
```
