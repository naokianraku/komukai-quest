# 小向クエスト ― あそびかた

東芝・小向工場を舞台にした、風刺アクションゲーム。
あなたは「**小向戦士**」。上司・部下・客・調達先ベンダー・スタッフ部門、そして数々の**裏切り**を乗り越え、出世の頂点を目指せ！

- ゲーム直リンク: **https://naokianraku.github.io/komukai-quest/game/**

インストール不要。スマホ・PCのブラウザでそのまま遊べます。

## 目的
**小向戦士 → 小隊長 → 大隊長 → 司令官**と出世しながら、各ステージのボスを倒してエンディングを目指す。

体力ゲージは「**仕事体力**」。0になると残機が1つ減ります。残機が尽きるとゲームオーバー。

## 操作

### 📱 スマホ（タッチ）
| やること | 操作 |
|---|---|
| 移動（4方向＋奥行き） | **画面の左半分をタッチ**すると、その場所が中心のスティックに。指を倒した方向へ動く |
| 攻撃（連打でコンボ） | 右下の「**攻撃**」ボタン |
| ジャンプ | 「攻撃」の真上の「**跳**」ボタン |
| 一時停止 | 画面上部の **❚❚** ボタン |
| 決定（メニュー） | 画面をタップ |

### 💻 PC（キーボード）
| やること | キー |
|---|---|
| 移動 | ← ↑ ↓ → または W A S D |
| 攻撃（連打でコンボ） | J または Z |
| ジャンプ | K または X |
| 一時停止 | P または Esc |

## 難易度（タイトル画面で選択）
- **ふつう** … 標準ルール（残機3）
- **かんたん** … 残機多め（残機6）＋**自分も飛び道具**（攻撃で工具を投げる）
- **地獄** … **雑魚敵が2倍**に増殖＋残機少なめ（残機2）。腕に自信があればどうぞ

## 世界観 ― 敵と「味方」
- **敵**：上司・部下・客・ベンダー・スタッフ部門、各種「審査会」「管理チーム」。
  攻撃や雑談で、頭上の吹き出しに「**言葉の暴力**」を浴びせてくる（「全体像が見えない！」「再DRだな」「該非判定はどこだ？」…）。
- **味方（PJメンバ）**：各ステージに3人。**でも、ボスの直前で全員が裏切って敵に回る！**
  さらに戦闘中の**うっかりミスで後ろから誤射**してくることも。信じられるのは自分だけ。

## 見分け方
- **作業着は全員水色＋黒の安全靴**。客・ベンダー・社長クラスは**スーツ**。
- **ヘルメットの色＋頭上の役職ラベル**で識別。自分は**橙の▼マーカー**。現在地は画面上部に表示。

---

## 攻略のヒント
- 敵は**予備動作**のあとに攻撃してくる。動きを見たら、離れるか先に殴る。
- **奥行き（上下）をずらす**と攻撃をかわせる（2.5D）。横一列に並ばないこと。
- **ジャンプ**で誘導弾（携行ミサイル兵のミサイル）を回避。
- 集団（徒党）に囲まれたら、画面の端へ動いて**各個撃破**。
- **ボス手前では味方が裏切る**。囲まれないよう位置取りを。
